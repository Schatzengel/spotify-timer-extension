# Spotify Web Timer

A clean and simple Chrome extension designed to automatically pause your Spotify music after a set duration. Perfect for falling asleep while listening to music.

## Features
- **Custom Timer:** Set any duration manually (in minutes).
- **Quick Presets:** Easily start a timer with one click (15m, 30m, 60m).
- **Cancel Timer:** Stop the countdown whenever you want.
- **Visual Feedback:** Real-time countdown display with a sleek, Spotify-inspired dark interface.
- **Minimalist Design:** Modern, distraction-free UI.

## Installation
1. Download this repository as a ZIP file and extract it to a folder on your computer.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top-right corner.
4. Click the **Load unpacked** button.
5. Select the folder where you extracted the project.

## How to Use
1. Click the extension icon in your browser toolbar.
2. Enter the number of minutes or select one of the presets.
3. Click **Start Timer**. 
4. The extension will automatically pause your Spotify tab once the timer reaches zero.

## Built With
- HTML5 & CSS3
- JavaScript (Chrome Alarms API)
